# みんレポ 差枚ランキングビューア

[min-repo.com](https://min-repo.com/) のデータを使い、**ホール名で検索 → その店の「全台 差枚ランキング」を台番号付きで最新10日分** まとめて表示するローカルWebアプリです。

## 起動方法

- `起動.bat` をダブルクリック、または
- コマンドで `python server.py`

起動するとブラウザが自動で開きます（http://127.0.0.1:8765 ）。
終了はコンソールで `Ctrl+C`。

> Python 3 だけで動きます（追加ライブラリのインストール不要）。

## 使い方

1. 検索ボックスにホール名を入力（例: `マルハン`、`センター`、`ジアスセンター南`）して検索。
2. 候補のホールがボタンで出るので選択。
3. 最新10日分のデータを取得し、2つのビューで表示します。
   - **日別ランキング**: 各日の全台を差枚順に表示（順位・台番・機種・差枚・G数・出率）。
   - **台番×日 差枚マトリクス**: 同じ台番（座席）の差枚を10日横断で比較。色が濃いほど差枚の絶対値が大きく、10日合計の降順で並びます。良い座席の傾向が見えます。

## 仕組み（技術メモ）

- min-repo はクエリ付きページ（`?kishu=all` など）を、`_d2` という Cookie が無いと空で返します。
  この `_d2` はトップ/レポートページの inline JS が設定する固定トークンで、本アプリは
  ページから自動抽出（取得失敗時は既知の値にフォールバック）して付与しています。
- 取得した各日のランキングは `cache/` に保存し、再表示を高速化します（既定6時間）。
- サーバへの連続アクセスには小さな待機を入れています。

## ファイル

- `server.py` … 本体（HTTPサーバ＋スクレイパ＋フロントエンドHTMLを内蔵）
- `cache/` … 取得データのキャッシュ
- `起動.bat` … Windows用の起動ショートカット
